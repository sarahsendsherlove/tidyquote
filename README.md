# TidyQuote — free tools & templates for house cleaning businesses

Free, no-signup tools for people starting or running a residential cleaning
business. Everything is plain HTML, loads fast, and works on any device —
hosted with GitHub Pages from this repo.

**Live site:** https://sarahsendsherlove.github.io/tidyquote/
